```bash
node cosmwasm/submit-proposal.js store -c Multisig -t "Upload Multisig contract v1.2.1" -d "Upload Multisig contract v1.2.1" -r $RUN_AS_ACCOUNT --deposit $DEPOSIT_VALUE --instantiateAddresses $INIT_ADDRESSES --version 1.2.1
```

2. Upgrade Multisig contract

There is no state migration needed during upgrade.

```bash
node cosmwasm/submit-proposal.js migrate \
  -c Multisig \
  -t "Migrate Multisig to v1.2.1" \
  -d "Multisig to v1.2.1" \
  --msg '{}' \
  --fetchCodeId \
  --deposit $DEPOSIT_VALUE
```
