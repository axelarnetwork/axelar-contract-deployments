## Background

This release introduces modular handler broadcasting support and adds event verifier functionality. Key improvements include:

- **Modular Handler Broadcasting**: Enables broadcasting for modular handlers, improving the architecture for handler-based operations
- **Event Verifier**: Adds comprehensive event verifier support with configurable attributes and event filtering
- **Handler Migrations**: Includes stellar and sui handler migrations
- **SDK 0.50 Support**: Backport support for SDK 0.50 upgrade
- **Multisig Handler**: Adds multisig module handler support
- **EVM Handler Improvements**: Unifies modular EVM handlers and adds message verifier handler
- **Bug Fixes**: Prevents gRPC server from hanging during shutdown

## Deployment

Restart `ampd` with the new binary. Binaries can be found at [axelar-amplifier/releases/tag/ampd-v1.14.0](https://github.com/axelarnetwork/axelar-amplifier/releases/tag/ampd-v1.14.0).

### Checklist

Run the below command to output the version:

```sh
$ ampd --version
ampd 1.14.0
```

Check `ampd` logs to ensure it restarts fine. Monitor voting and signing for your verifier on axelarscan to verify it's operating correctly.
